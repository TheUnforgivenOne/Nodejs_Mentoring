### Repository for Node.js mentoring program

#### API
By default port: 3000

GET '/' - Home page

GET '/users' - users list

POST '/users?login="string"&password="string"&age="number"' - create new user

GET '/users/:id' - user by id

PUT '/users/:id?login="string"&password="string"&age="number"' - update user by id

DELETE '/users/:id' - delete user by id
